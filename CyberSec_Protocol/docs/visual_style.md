# Визуальный стиль и атмосфера

## 3D-окружение

### Геометрия и модели
- **Low-poly стиль:** Умеренно детализированные модели с акцентом на узнаваемость форм
- **Серверные стойки:** Тяжёлые металлические конструкции высотой 2–3 метра
- **Оборудование:**
  - Ленточные накопители (DAT, DLT)
  - Патч-панели с цветными кабелями
  - Коммутаторы и маршрутизаторы эпохи Cisco 2900/3600
  - ИБП и распределительные щиты
- **Декорации:**
  - Кабельные лотки под потолком и под фальшполом
  - Огнетушители, знаки безопасности
  - Тележки для оборудования
  - Кондиционеры и вентиляционные решётки

### Освещение
- **Основной свет:** Мерцающие люминесцентные лампы дневного света
- **Акцентное освещение:**
  - Индикаторы на серверах (зелёные, жёлтые, красные светодиоды)
  - Экраны ЭЛТ-мониторов как источники света
  - Аварийное красное освещение в режиме тревоги
- **Динамические эффекты:**
  - Случайные мерцания ламп
  - Изменение интенсивности при перегрузке сети
  - Тени от движущихся объектов (игрок, тележки)

## CRT-эффекты интерфейса

### Обязательные эффекты
- **Искривление краёв:** Бочкообразная дисторсия по периметру экрана
- **Хроматическая аберрация:** Разложение цвета на краях контрастных объектов
- **Строчная развёртка:** Горизонтальные полосы, имитирующие refresh CRT
- **Шум:** Лёгкий зернистый фильтр поверх изображения
- **Задержка отклика:** 1–2 кадра между действием и визуальной реакцией

### Настройки для комфорта
- Ползунок интенсивности эффектов (0–100%)
- Возможность полного отключения каждого эффекта отдельно
- Режим «Современный монитор» для игроков с чувствительностью к мерцанию

## Терминал

### Внешний вид
- **Позиционирование:** Полупрозрачный оверлей или отдельный 3D-монитор в локации
- **Шрифт:** Моноширинный, стилизованный под эпоху (Terminus, Fixedsys, VT323)
- **Цветовая палитра:**
  - Зелёный на чёрном (классика)
  - Янтарный на чёрном (альтернатива)
  - Белый на синем (режим некоторых систем)

### Функциональные элементы
- **История команд:** Прокрутка вверх/вниз, поиск по истории
- **Автодополнение:** Tab-completion для команд и путей
- **Подсветка синтаксиса:** Цветовое выделение команд, параметров, путей
- **Встроенная справка:** Команда `help` с контекстными подсказками
- **Многозадачность:** Поддержка нескольких вкладок или сессий

### Анимации
- Появление текста посимвольно (настраиваемая скорость)
- Эффект «разогрева» монитора при включении
- Помехи при плохом соединении или взломе

## Звуковое сопровождение

### Фоновые звуки
- Гул серверных кулеров (меняется от нагрузки)
- Щелчки реле и переключателей
- Звук печатающегося текста
- Шум жёстких дисков (характерный треск IDE/SATA)

### Интерфейсные звуки
- Клики клавиш терминала
- Звуки успешных/неуспешных команд
- Сигналы тревоги при обнаружении
- Звук модемного соединения (для ностальгии)

### Динамическое аудио
- Изменение гула при перегреве оборудования
- Нарастание звука сирены при эскалации тревоги
- Приглушение фоновых звуков при фокусе на терминале

## Цветовая палитра

### Основные цвета
```
Тёмно-серый: #1a1a1a (фоны, корпуса)
Серо-зелёный: #2d3e30 (стены ЦОД)
Электрический синий: #0066cc (акценты, кабели данных)
Предупреждающий жёлтый: #ffcc00 (индикаторы, предупреждения)
Тревожный красный: #cc0000 (сигнализация, ошибки)
Терминальный зелёный: #00ff00 (основной цвет интерфейса)
```

### Акцентные цвета для кабелей
```
Синий: сеть данных
Жёлтый: оптоволокно
Красный: критические соединения
Зелёный: заземление
Серый: питание
```
