# Уровни и локации

## Список локаций

### Тир 1: Начальные локации (Миссии 1–10)

---

#### 1. Офис малого бизнеса «TechStart LLC»
**Тип:** Обучение  
**Размер:** Малый (1 комната)  
**Сложность:** Очень низкая

**Описание:**
Небольшой офис стартапа с 5–6 рабочими местами, одним серверным шкафом в углу и старым ЭЛТ-монитором. Идеальное место для первых шагов.

**Ключевые точки:**
- Рабочие станции (3–5 шт.)
- Серверный шкаф (1 стойка)
- Рабочее место администратора
- Принтер/сканер

**Задачи:**
- Сканирование локальной сети
- Поиск слабых паролей
- Установка патчей

**Уникальные особенности:**
- Нет охраны
- Можно ошибаться без последствий
- Подробные подсказки

**Награды:**
- Базовое оборудование
- Первые очки репутации
- Доступ к следующей локации

#### 2. Городской банк «First National»
**Тип:** Аудит безопасности  
**Размер:** Средний (3 комнаты)  
**Сложность:** Низкая

**Описание:**
Отделение банка с клиентской зоной, серверной комнатой и офисом менеджера. Система безопасности базового уровня.

**Ключевые точки:**
- Клиентская зона (публичный доступ)
- Серверная комната (требуется ключ-карта)
- Офис менеджера (пароль на двери)
- Банкомат (внешний периметр)

**Задачи:**
- Проверка фаервола периметра
- Аудит рабочих станций сотрудников
- Тестирование физической безопасности

**Уникальные особенности:**
- Присутствуют NPC-сотрудники
- Ограничение по времени (рабочие часы)
- Камеры наблюдения (можно отключить)

**Награды:**
- Кредиты (средние)
- Чертеж базового фаервола
- Контакт для будущих миссий

## Генерация процедурных локаций

### Для ежедневных контрактов

**Параметры генерации:**

| Параметр | Варианты | Влияние |
|----------|----------|---------|
| **Тип здания** | Офис, склад, больница, школа, завод | Структура комнат |
| **Размер** | S, M, L, XL | Количество узлов |
| **Безопасность** | Низкая, средняя, высокая, экстремальная | Типы препятствий |
| **Время суток** | День, ночь, выходные | Количество NPC |
| **Погода** | Ясно, дождь, снег | Влияние на физические задачи |

### Алгоритм генерации

```
1. Выбрать тип здания → определить шаблон планировки
2. Определить размер → масштабировать количество комнат
3. Расставить ключевые точки (цель, вход, серверная)
4. Добавить препятствия (охрана, камеры, замки)
5. Разместить лут и находки лора
6. Настроить параметры сети (узлы, фаерволы)
7. Добавить модификаторы (если ежедневный контракт)
```

### Примеры модификаторов

- **«Зеркало»** — все узлы сети имеют двойников
- **«Таймер»** — обратный отсчёт с начала миссии
- **«Слепота»** — мини-карта отключена
- **«Призрак»** — одно обнаружение = провал
- **«Берсерк»** — удвоенная скорость, но удвоенный риск

## Дизайн уровней

### Принципы проектирования

1. **Три пути прохождения**
   - Стелс (скрытное проникновение)
   - Штурм (агрессивный взлом)
   - Социальный (через NPC и обман)

2. **Петля обратной связи**
   - Каждое действие имеет последствия
   - Игрок видит результат своих решений
   - Возможность вернуться и исправить ошибки

3. **Вертикальность**
   - Многоуровневые пространства
   - Разные пути на разных высотах
   - Контроль высоты = контроль ситуации

4. **История через окружение**
   - Каждая локация рассказывает историю
   - Записки, логи, визуальные подсказки
   - Игрок сам собирает нарратив

### Чеклист дизайна уровня

- [ ] Чёткая цель миссии
- [ ] Минимум 2 пути к цели
- [ ] Возможности для стелса
- [ ] Точки сохранения/чекпоинты
- [ ] Награды за исследование
- [ ] Элементы лора
- [ ] Баланс сложности
- [ ] Тестирование на проходимость

## Быстрое перемещение

### unlocked точки

После первого посещения игрок может быстро перемещаться между:
- Своим офисом (хаб)
- Посещёнными безопасными локациями
- Контактами (Мария, Дима, Сара)

### Ограничения

- Нельзя быстрое перемещение во время активных миссий
- Некоторые локации блокируются после завершения сюжета
- Быстрое перемещение стоит кредиты (расходы на транспорт)
