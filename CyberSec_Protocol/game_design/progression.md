# Система прогрессии

## Дерево навыков

### Общая структура

Дерево навыков разделено на 5 основных веток, каждая из которых содержит 3 уровня мастерства:
- **Новичок** (1–2 очка навыка)
- **Специалист** (3–4 очка навыка)
- **Эксперт** (5+ очков навыка)

```
                        [СТАРТ]
                          │
          ┌───────────────┼───────────────┐
          │               │               │
   [Сетевая        [Криптография]   [Аппаратное
    разведка]                        обеспечение]
          │               │               │
          └───────────────┼───────────────┘
                          │
                  [Социальная
                   инженерия]
                          │
                  [Инцидент-
                   менеджмент]
```

## Ветка 1: Сетевая разведка

### Описание

## Система репутации

### Уровни репутации

| Уровень | Диапазон | Преимущества |
|---------|----------|--------------|
| **Неизвестный** | 0–500 | Базовые контракты, стандартное оборудование |
| **Начинающий** | 501–1500 | Доступ к обучению, скидки 5% |
| **Опытный** | 1501–3500 | Контракты среднего уровня, скидки 10% |
| **Профессионал** | 3501–6000 | Премиум контракты, скидки 15%, уникальные предметы |
| **Легенда** | 6001–10000 | Эксклюзивные миссии, скидки 20%, доступ к бета-тестам |

### Факторы влияния на репутацию

**Положительные:**
- Успешное завершение миссий (+50–500)
- Стиль «Призрак» (без обнаружения) (+100)
- Помощь другим игрокам (кооператив) (+50)
- Открытие уязвимостей и ответственный доклад (+200)

**Отрицательные:**
- Провал миссии (-50–200)
- Обнаружение и поимка (-150)
- Повреждение критических систем (-100)
- Нарушение контракта (-300)

## Баланс и прогрессия

### Кривая сложности

```
Миссии 1–5:   Обучение, простые задачи, низкий риск
Миссии 6–15:  Первые вызовы, средние награды
Миссии 16–30: Сложные головоломки, высокие ставки
Миссии 31+:   Элитные контракты, уникальные награды
```

### Рекомендуемый темп прогрессии

| Часов игры | Ожидаемый уровень репутации | Открыто веток навыков |
|------------|----------------------------|----------------------|
| 1–3 часа   | 0–500                      | 1 ветка (новичок)    |
| 5–10 часов | 500–2000                   | 2–3 ветки (специалист) |
| 15–25 часов| 2000–5000                  | 4–5 веток (эксперт)  |
| 30+ часов  | 5000–10000                 | Все ветки макс       |
