# UI/UX Дизайн

## Общие принципы

### Философия интерфейса

1. **Иммерсивность:** Интерфейс не должен вырывать игрока из атмосферы 2000-х
2. **Функциональность:** Все необходимые данные доступны без лишних кликов
3. **Ностальгия:** Эстетика ранних ОС и терминалов
4. **Доступность:** Настройки для игроков с ограниченными возможностями

### Цветовая схема терминала

**Основная палитра:**
```
Фон: #0a0a0a (почти чёрный)
Текст: #00ff00 (классический зелёный)
Акцент: #00cc00 (тёмно-зелёный для выделения)
Ошибка: #ff3333 (красный)
Предупреждение: #ffcc00 (жёлтый)
Инфо: #0099ff (синий)
Успех: #00ff66 (светло-зелёный)
```

**Альтернативные темы:**
- **Янтарь:** #ffb000 на #1a0f00
- **Белый на синем:** #ffffff на #0000aa
- **Монохром:** оттенки серого

**Элементы HUD:**

#### Верхняя панель
- **Вычислительные циклы:** Прогресс-бар + числовое значение
- **Пропускная способность:** Прогресс-бар + Мбит/с
- **Уровень обнаружения:** Прогресс-бар + %, меняет цвет (зелёный → жёлтый → красный)
- **Мини-карта:** Схематичное изображение текущей зоны

#### Терминал (нижняя панель)
- Поле ввода команд
- История вывода (скролл)
- Кнопки быстрых действий:
  - **HIST** — история команд
  - **AUTOCOMPLETE** — подсказки
  - **HELP** — справка

**Настройки HUD:**
- Размер терминала (компактный/полный)
- Прозрачность панелей
- Отключение мини-карты
- Скорость прокрутки текста

## Интерактивные элементы

### Кнопки

**Стили:**
```
[ СТАНДАРТНАЯ ]     — прямоугольная, зелёная рамка
[ АКТИВНАЯ ]        — заполненная зелёным, белый текст
[ ЗАБЛОКИРОВАНА ]   — серая, полупрозрачная
[ ОПАСНАЯ ]         — красная рамка (удаление, выход)
```

**Анимации:**
- Наведение: увеличение яркости рамки
- Нажатие: эффект «вдавливания»
- Блокировка: пульсация серым

### Ползунки

**Использование:**
- Настройка громкости
- Интенсивность CRT-эффектов
- Чувствительность мыши
- Скорость текста терминала

**Вид:**
```
Громкость: [████████░░] 80%
            ↑ (можно перетаскивать)
```

### Выпадающие списки

**Пример:**
```
Выберите тему: [ Зелёная ▼ ]
                 ┌─────────────┐
                 │ Зелёная     │
                 │ Янтарная    │
                 │ Синяя       │
                 │ Монохром    │
                 └─────────────┘
```

## Анимации и переходы

### Переходы между экранами

- **Появление:** Постепенное проявление (fade-in)
- **Исчезновение:** Плавное затухание (fade-out)
- **Смена вкладок:** Горизонтальный слайд

### Текстовые эффекты

- **Печать:** Посимвольное появление (настраиваемая скорость)
- **Глюк:** Случайные символы при ошибках
- **Мерцание:** Для важных уведомлений

### Уведомления

**Типы:**
```
┌──────────────────────────┐
│ ✓ Успешно                │  — зелёный фон
│ Скрипт выполнен          │
└──────────────────────────┘

┌──────────────────────────┐
│ ⚠ Внимание               │  — жёлтый фон
│ Низкий заряд батареи     │
└──────────────────────────┘

┌──────────────────────────┐
│ ✗ Ошибка                 │  — красный фон
│ Соединение разорвано     │
└──────────────────────────┘
```

**Позиция:** Правый верхний угол
**Длительность:** 3–5 секунд
**Максимум одновременно:** 3 уведомления

## Адаптивность и доступность

### Настройки для комфорта

| Опция | Описание |
|-------|----------|
| **Размер шрифта** | Малый / Средний / Крупный / Огромный |
| **CRT-эффекты** | Ползунок интенсивности 0–100% |
| **Цветовой режим** | Для дальтоников (протанопия, дейтеранопия) |
| **Скорость текста** | 50% – 200% от базовой |
| **Автосохранение** | Вкл/Выкл |
| **Подсказки** | Вкл/Выкл/Только обучение |

### Управление

**Клавиатура:**
- `Tab` — автодополнение команд
- `Стрелки вверх/вниз` — история команд
- `Ctrl+C` — прерывание текущей операции
- `F1` — встроенная справка
- `Esc` — закрыть окно/меню

**Мышь:**
- Левая кнопка — выбор, клик
- Правая кнопка — контекстное меню
- Колесо — прокрутка терминала
- Drag-and-drop — в инвентаре и редакторе скриптов

**Геймпад (опционально):**
- Поддержка Xbox/PlayStation контроллеров
- Виртуальная клавиатура для ввода команд
- Быстрые слоты для скриптов

## Прототипирование

### Инструменты

- **Figma** — дизайн макетов
- **Adobe After Effects** — анимации переходов
- **Unity UI Toolkit** — реализация в движке

### Тестирование UX

1. **Первый запуск:** Онбординг для новых игроков
2. **Юзабилити-тесты:** 5–10 игроков, сбор фидбека
3. **A/B тестирование:** Сравнение вариантов интерфейса
4. **Метрики:** Время на задачу, количество ошибок, удовлетворённость

### Итерации дизайна

```
Версия 0.1 → Базовые экраны
Версия 0.3 → Добавление анимаций
Версия 0.5 → Учёт фидбека, улучшение читаемости
Версия 0.7 → Оптимизация производительности
Версия 1.0 → Финальный полиш
```
